# Skills

This directory groups the public skill collection into three architecture bundles.

## Bundles

- `lifecycle-methodology/` — architecture lifecycle routing and staged delivery
- `diagramming/` — architecture diagram methods and artifact-specific drawing guidance
- `architecture-supplements/` — cross-cutting architecture judgment and comparison skills

## Packaging rule

One skill = one folder.

Within each skill folder:
- `SKILL.md` is the entrypoint
- support files stay beside the skill
- copy the whole folder when installing into another environment
